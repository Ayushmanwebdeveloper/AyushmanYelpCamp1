There are two types of campgrounds: Private and public. Both have their advantages but finding a good one can be hard. That is why it's a good idea to use campground reviews and ratings to guide you. Resources like AAA and Trailer Life compile all the information you need to see if a campground is right for you.

SpringCamp is a website created by Ayushman Tripathi where users can create and review campgrounds. In order to review or create a campground, you must have an account. This project was part of Colt Steele's web dev course on udemy. This project is created using Node.js, Express, MongoDB, and Bootstrap. Passport.js was used to handle authentication.

  • Users can create, edit, and remove campgrounds
  • Users can review campgrounds once, and edit or remove their review
  • User profiles include more information on the user (full name, email, phone, join date), their campgrounds, and the option to edit their profile or delete their account
  • Search campground by name or location
  • Sort campgrounds by highest rating, most reviewed, lowest price, or highest price
